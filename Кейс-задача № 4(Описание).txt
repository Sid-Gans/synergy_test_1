
Проектирование архитектуры:
  Выбор ISAPI-модуля как оптимального решения для Delphi
  Разделение на клиентскую (HTML+JS) и серверную (Delphi) части
  Использование Bootstrap для быстрого прототипирования интерфейса

Реализация функционала:
  Создание модуля WebBroker в Delphi
  Настройка соединения с MS SQL Server через dbExpress
  Реализация REST-like API для работы с заказами

Особенности реализации:
  Генерация JSON на сервере (без использования сторонних библиотек)
  Простая маршрутизация запросов через Actions
  Использование подготовленных SQL-запросов для безопасности

Серверная часть (Delphi):
  WebModuleUnit.pas - основной модуль обработки запросов
  SQLConnection - подключение к базе данных
  TSQLQuery - выполнение SQL-запросов
  TJSONObject/TJSONArray - формирование JSON-ответов

Клиентская часть:
  Bootstrap 5 - для базового интерфейса
  Простые HTML-страницы с ссылками на API

База данных:
  3 основные таблицы (клиенты, услуги, заказы)
  Внешние ключи для обеспечения целостности данных
